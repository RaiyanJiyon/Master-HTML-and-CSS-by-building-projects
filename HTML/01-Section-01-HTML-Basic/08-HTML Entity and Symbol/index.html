HTML Entity Symbols

Here are some common HTML entity symbols:

  • © - Copyright symbol: &copy;
  • ® - Registered trademark symbol: &reg;
  • ™ - Trademark symbol: &trade;
  •   - Non-breaking space: &nbsp;
  • < - Less than symbol (left angle bracket): &lt;
  • > - Greater than symbol (right angle bracket): &gt;
  • & - Ampersand symbol: &amp;
